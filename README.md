### For the correct operation of the games you will need: Python v3.6 and higher, Poetry v1.2.0 and higher


### How to install: 
Git clone https://github.com/vasiliyMatrosov/python-project-49

make setup

make build

make package-install


### How to play:
insert the commands:

brain-even 

crain-calc

brain-gcd

brain-progression

brain-prime
